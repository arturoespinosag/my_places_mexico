# My Places Mexico

A Flutter application to save and manage favorite places with geolocation support.

## Features
- Save places with location data
- Map integration
- Persistent storage

## Architecture
- Clean Architecture (presentation / domain / data)
- State Management: BLoC
- Networking: Dio

## Tech Stack
- Flutter
- BLoC
- Firebase integration


## Key Learnings
- Handling geolocation in Flutter
- Managing app state at scale
